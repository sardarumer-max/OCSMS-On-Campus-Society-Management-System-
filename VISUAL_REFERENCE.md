## Desktop Display Preview

Your OCSMS application will look like this on any macOS, Windows, or Linux computer:

1. **Login Screen**: Clean, dark blue interface with mint-green buttons
2. **Dashboard**: Multiple cards with statistics in cyan text
3. **Data Tables**: Dark rows alternating with slightly lighter rows
4. **Navigation**: Consistent teal borders and cyan highlights
5. **Forms**: Dark input fields with teal outlines and cyan caret

All text will be clearly readable with high contrast, and color blind users will still be able to distinguish different elements since the palette doesn't rely solely on color differentiation.
